# Deploy Your Presentation to the Web

Your Keynote presentation is ready to be deployed! Here are the easiest options:

## Option 1: Netlify Drop (Easiest - No Account Needed)

1. Go to https://app.netlify.com/drop
2. Drag and drop the entire `Design_Nubank_v1.1` folder onto the page
3. Wait for upload to complete
4. You'll get a URL like `https://random-name-123.netlify.app`
5. Share this URL with anyone!

**Note:** The free URL will be random. For a custom domain, you'll need a Netlify account.

## Option 2: GitHub Pages (Free with Custom URL)

1. Create a GitHub account (if you don't have one)
2. Create a new repository on GitHub
3. Upload all files from this folder to the repository
4. Go to Settings > Pages
5. Select the main branch and save
6. Your site will be available at `https://yourusername.github.io/repository-name`

## Option 3: Vercel (Free, Easy)

1. Go to https://vercel.com
2. Sign up/login with GitHub
3. Click "Add New Project"
4. Import your GitHub repository (or drag and drop the folder)
5. Deploy - you'll get a URL instantly!

## Option 4: Surge.sh (Command Line - Free)

If you have Node.js installed:
```bash
npm install -g surge
cd /Users/lavisantos/Desktop/Design_Nubank_v1.1
surge
```
Follow the prompts to get a free URL.

---

**Recommended:** Use Option 1 (Netlify Drop) for the fastest deployment without any setup!
